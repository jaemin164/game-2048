# 2048 게임

웹 브라우저에서 즐길 수 있는 클래식 2048 퍼즐 게임입니다.

## 게임 방법

- **방향키**를 사용하여 타일을 움직입니다
- 같은 숫자의 타일이 만나면 합쳐집니다
- **2048 타일**을 만들면 승리!
- 더 이상 움직일 수 없으면 게임 오버

## 실행 방법

1. `index.html` 파일을 웹 브라우저에서 엽니다
2. 방향키로 게임을 시작하세요!

## 기능

- 점수 및 최고 점수 저장
- 반응형 디자인 (모바일 지원)
- 부드러운 애니메이션
- 로컬 스토리지를 통한 최고 점수 기억

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
